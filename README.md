## Sobre este proyecto

Portafolio profesional interactivo de un solo archivo, diseñado para展示 las competencias en **Sistemas de Información Geográfica (SIG)**, **Ordenamiento Territorial** y **Gestión del Riesgo de Desastres**.

**En vivo:** [Ver portafolio](https://wolfssohn.github.io/portafolio/)

---

## Características

- **Single-page** — Un solo `index.html`, sin dependencias de frameworks
- **Dark theme** — Diseño oscuro con acentos púrpura/esmeralda/fucsia
- **Responsivo** — Adaptado para desktop, tablet y móvil
- **Animaciones** — Scroll-reveal y efecto de máquina de escribir
- **Canvas interactivo** — Fondo topográfico con cuadrícula UTM, brújula y barra de escala
- **Overlay de coordenadas** — Mapeo del cursor al territorio de Chile
- **Formulario de contacto** — Integrado con Formspree

## Despliegue local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/wolfssohn/portafolio.git
   cd portafolio
   ```

2. Abre `index.html` en tu navegador o usa un servidor local:
   ```bash
   # Con Python
   python -m http.server 8000

   # Con Node.js
   npx serve .
   ```

3. Visita `http://localhost:8000`
